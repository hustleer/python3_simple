############# Progress Bar ####################<br>
-To show progress of current process.<br>
USAGE:<br>
  syntax: progress(current,total,barlength)<br>
          barlength default = 65<br>
  
  example: <br>
    for i in range(1,100): <br>
        progress(i,100,20) <br>
